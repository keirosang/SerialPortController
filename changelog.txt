SerialPortCollector 开发日志

v1.0.2 (2024-01-19)
新增功能：
1. 添加 TCP 数据透传功能
   - 支持配置 TCP 服务器地址和端口
   - 支持自动重连机制
   - 添加 2 秒发送超时机制
   - 超时自动清空发送队列，避免数据积压

2. 优化状态显示逻辑
   - 修改状态判断机制，更准确反映设备状态
   - Active: 有数据接收时显示
   - Waiting: 初始状态和无数据时显示
   - Offline: 超过配置的超时时间无数据时显示

3. 添加错误日志系统
   - 创建 error 目录存储错误日志
   - 按日期生成日志文件（YYYYMMDD.log）
   - 记录详细的错误信息和时间戳
   - 错误信息不再显示在控制台，改为写入日志文件

4. 数据速率显示优化
   - 添加实时数据速率统计
   - 无数据时自动清零速率显示
   - 显示更精确的字节/秒统计

5. 配置文件扩展
   - 添加 TCP 转发相关配置
   - 添加超时时间配置
   - 添加重连间隔配置

改进：
1. 控制台显示优化
   - 移除多余的状态信息
   - 简化界面布局
   - 优化颜色显示逻辑

2. 性能优化
   - 优化数据处理逻辑
   - 减少不必要的控制台输出
   - 添加数据队列管理

3. 错误处理增强
   - 添加更多错误检查
   - 完善错误恢复机制
   - 改进错误日志记录

4. 代码结构优化
   - 添加 TcpClient 类
   - 添加 Logger 类
   - 优化线程同步机制

修复：
1. 修复状态显示不稳定的问题
2. 修复数据统计可能不准确的问题
3. 修复内存泄漏隐患
4. 修复类型转换警告

对比 v1.0.1：
1. 新增 TCP 数据转发功能
2. 新增错误日志系统
3. 改进状态显示机制
4. 优化性能和稳定性
5. 完善错误处理机制
